How to use 
Camera mode (Cardputer Flash TX → web):
  1. "Start camera" and allow access
  2. Aim camera so the Cardputer LCD fills the yellow ROI box
  3. On Cardputer (Flash TX): type a message, press Enter
  4. The decoder auto-calibrates the unit time from the first few pulses

Microphone mode (Cardputer Audio TX → web):
  1. "Start microphone" and allow access
  2. Be in the same room as the Cardputer (or point its speaker at you)
  3. On Cardputer (Audio TX): type, press Enter
  4. Adjust "tone Hz" if your transmitter uses a different sidetone

Transmit panel (web → Cardputer mic):
  1. On Cardputer: Tab to "Audio RX"
  2. In the web Transmit panel: type, click Send
  3. The web plays a 700 Hz tone train; the Cardputer decodes it

Page must be served from https:// or localhost (camera/mic permission rule).
